# 本项目声明

* 兴趣使然的补充官方汉化的缺失和疏漏
* 优先追求翻译进度，其次保证准确率
* 主要在 Google / Microsoft 翻译的基础上润色
* 欢迎任何人 推送 / 拉取 / 合作 二次翻译

## 手动替换官方语言包

* 下载 [语言文件](https://github.com/millionart/Blender_Chinese_localization_Unofficial/raw/master/zh_CN/zh_CN.mo)
* 改名为 `blender.mo`
* 替换 `Blender\2.80\datafiles\locale\zh_CN\LC_MESSAGES` 同名文件

    注意 **建议隔一段时间进行一次更新**
